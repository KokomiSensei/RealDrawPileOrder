# Real Draw Pile Order

[English version](./READEME.md) | [简体中文版本](./README.zh-CN.md)

## Description

This is a mod for the game "Slay the Spire 2" that allows players to see the order of cards in their draw pile, making it easier for players to strategize and plan their moves.

This mod doesn't require other players in multiplayer to have it installed. This is made possible by setting the "affects_gameplay" property to "false" in the mod's [manifest json](./RealDrawPileOrder.json).

## Compatibility

This mod is developed and tested on the v0.99.1 version of the game, and may not be compatible with future versions. Please check the mod's publish page for updates and compatibility information.

## Installation, Usage

See the mod's publish page at [Nexus Mods](https://www.nexusmods.com/slaythespire2/mods/414)

## Building from source

1. Download Slay the Spire 2
2. Install .NET 9.0 SDK
3. Clone the repository
4. Cd into the project directory
5. Run `dotnet build -c Release` for release build or `dotnet build -c Debug` for debug build
6. The mod will be compiled and copied to the `mods` folder in the Slay the Spire 2 installation directory
7. Launch the game and see the mod in action!
